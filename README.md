# Проект аналитики финансовых транзакций

## Описание

Проект реализует основные функции для генерации JSON-ответов, которые используются веб-страницами для отображения данных по транзакциям, валютам и акциям.

---

## Структура

- Основные функции генерации JSON-ответов находятся в модуле `views.py`.
- Данные для анализа берутся с начала месяца, на который выпадает входящая дата, по саму входящую дату.
- Настройки пользователя (`user_currencies` и `user_stocks`) хранятся в JSON-файле `user_settings.json`.
- Для получения курсов валют и цен на акции используется библиотека `requests` и сторонние API.

---
## Логирование

- Декоратор без параметров — записывает данные отчёта в файл с именем по умолчанию.
- Декоратор с параметром — принимает имя файла для записи.

---

## Тестирование

- Покрытие тестами функционального кода — 80% и выше.
- Тесты написаны с использованием `pytest`.
- В тестах используются:
  - `Mock` и `patch` для заглушек и подмены.
  - Фикстуры для генерации тестовых данных.
  - Параметризация тестов для разных входных данных.
- Все тесты проходят успешно при запуске командой `pytest`.

---

## Используемые библиотеки и технологии

- `json` — для работы с JSON-ответами.
- `requests` — для обращения к внешним API валют и акций.
- `datetime` — для работы с датами.
- `logging` — для логирования действий.
- `pytest` — для тестирования.
- `pandas` — для обработки данных транзакций.
